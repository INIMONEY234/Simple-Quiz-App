# Simple-Quiz-App

# 🧠 Quiz App

A simple multiple-choice Quiz Application built with HTML, CSS, and JavaScript.  
This project helps practice core frontend concepts such as DOM manipulation, arrays, objects, and application state management.

---

## 📌 Features

- Display one question at a time
- Multiple-choice answer selection
- Tracks user score in real time
- Progress indicator (e.g. Question 2 of 10)
- Final results screen showing total score
- Option to restart the quiz

---

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript (ES6)

---

## 📁 Project Structure
quiz-app/
│
├── index.html # Main HTML structure
├── style.css # Styling for the app
└── script.js # Quiz logic and interactivity


---

## 🚀 How to Run the Project

1. Clone or download the project folder
2. Open the folder in your code editor (e.g. VS Code)
3. Open `index.html` in your browser
4. Start answering questions!

---

## 🧠 Key Concepts Practiced

- Arrays and Objects (for storing questions)
- DOM Manipulation (updating UI dynamically)
- Event Handling (button clicks)
- Conditional Logic (checking answers)
- State Management (tracking score and progress)
- Dynamic UI Rendering
